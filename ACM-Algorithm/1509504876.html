高精度之高精除以高精

发表于 2017-04-08 更新于 2021-12-30 分类于 ACM/Algorithm Valine：

Description

高精除以高精

Input

每组测试数据个栈2行，每行代表一个高精度整数（不超过100位）

Output

每组测试数据输出占2行，分别输出商和余数。

Sample Input

140
5

Sample Output

28
0

思路:

高精除以高精是用减法模拟除法，对被除数的每一位都减去除数，一直减到当前位置的数字（包含前面的余数）小于除数（由于每一位的数字小于10，所以对于每一位最多进行10次计算）.

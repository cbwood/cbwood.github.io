Yet Another Number Sequence CodeForces - 392C 矩阵快速幂

发表于 2017-08-19 更新于 2021-12-30 分类于 ACM/Algorithm Valine：

题目链接:点我

题目

Everyone knows what the Fibonacci sequence is. This sequence can be defined by the recurrence relation:
F1 = 1, F2 = 2, Fi = Fi - 1 + Fi - 2 (i > 2).

We'll define a new number sequence Ai(k) by the formula:
Ai(k) = Fi × i^k (i ≥ 1).

In this problem, your task is to calculate the following sum: A1(k) + A2(k) + ... + An(k). The answer can be very large, so print it modulo 1000000007 (109 + 7).

Input

The first line contains two space-separated integers n, k (1 ≤ n ≤ 1e17; 1 ≤ k ≤ 40).

Output

Print a single integer — the sum of the first n elements of the sequence Ai(k) modulo 1000000007 (1e9 + 7).

Example

Input

1 1

Output

1

Input

4 1

Output

34

Input

5 2

Output

316

Input

7 4

Output

73825

题意:

如果提上公式所说:计算 A1(k) + A2(k) + … + An(k), Ai(k) = Fi × i^k.

思路:

矩阵快速幂.
首先我们应该知道,(n+1)k(n+1)^k(n+1)k =$ k \choose k$$ n^k$ +$ k \choose k-1$ nk−1n^{k-1}nk−1 + $ k \choose k-2$$n^{k-2}$ + … +$ k \choose 1$ n +$ k \choose 0$ n0n^0n0
我们令u(n+1,k) = (n+1)k(n+1)^k(n+1)k * F(n+1), v(n+1,k) = (n+1)k(n+1)^k(n+1)k * F(n)
根据斐波那契数列递推式:
u(n+1,k) = (n+1)k(n+1)^k(n+1)k * F(n+1)
= (n+1)k(n+1)^k(n+1)k * F(n) + (n+1)k(n+1)^k(n+1)k * F(n-1)
= ∑i=0k\sum_{i=0}^k∑i=0k​ (ki)k \choose i(ik​) * nin^ini * F(n) + ∑i=0k\sum_{i=0}^k∑i=0k​ (ki)k \choose i(ik​) * nin^ini * F(n-1)
= ∑i=0k\sum_{i=0}^k∑i=0k​ (ki)k \choose i(ik​) * nin^ini * F(n) + ∑i=0k\sum_{i=0}^k∑i=0k​ (ki)k \choose i(ik​) * v(n,i)
v(n+1,k) = (n+1)k(n+1)^k(n+1)k * F(n)
= ∑i=0k\sum_{i=0}^k∑i=0k​ (ki)k \choose i(ik​) * nin^ini * F(n)

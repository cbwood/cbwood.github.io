M斐波那契数列 HDU - 4549 矩阵快速幂

发表于 2017-08-23 更新于 2021-12-30 分类于 ACM/Algorithm Valine：

题目链接

题目

 M斐波那契数列F[n]是一种整数数列，它的定义如下：

F[0] = a
F[1] = b
F[n] = F[n-1] * F[n-2] ( n > 1 )

现在给出a, b, n，你能求出F[n]的值吗？

Input

输入包含多组测试数据；
每组数据占一行，包含3个整数a, b, n（ 0 <= a, b, n <= 10^9 ）

Output

对每组测试数据请输出一个整数F[n]，由于F[n]可能很大，你只需输出F[n]对1000000007取模后的值即可，每组数据输出一行。

Sample Input

0 1 0
6 10 2

Sample Output

0
60

题意:

emmm,中文题,不解释,

思路:

矩阵快速幂,

分析一下题目给的递推式:
f[0] = a;
f[1] = b;
f[2] = a * b;
f[3] = a∗b2a * b^{2}a∗b2;
f[4] = a2∗b3a^ 2 * b^{3}a2∗b3;
f[5] = a3∗b5a ^ 3 * b^{5}a3∗b5;
f[6] = a5∗b8a ^ 5 * b^{8}a5∗b8;
有没有发现第n项的 a, 和 b 的幂恰好是斐波那契数列的第 n - 2,和 n - 3 项.于是快速幂加快速幂就可以了.

其实我们可以发现最后化简出来(A+B)n=a+b∗B(A+\sqrt B)^n = a + b * \sqrt B(A+B​)n=a+b∗B​

(A−B)n=a−b∗B(A-\sqrt B)^n = a - b * \sqrt B(A−B​)n=a−b∗B​

我们可以得到最后的ans=bBans = b \sqrt Bans=bB​，那么我们只需要算一个b就可以了，这里算b除了上面题解说的那种分治的思想其实还可以用矩阵快速幂计算。

我们假设(A+B)n=an+bnB(A+ \sqrt B)^n = a_n + b_n \sqrt B(A+B​)n=an​+bn​B​

那么(A+B)n+1=an+1+bn+1B=(an+bnB)∗(A+B)=(A∗an+bn∗B)+(an+A∗bn)B(A+ \sqrt B)^{n+1} \\ = a_{n+1} + b_{n+1} \sqrt B \\ = (a_n + b_n\sqrt B) \ast (A+\sqrt B)\\ = (A \ast a_n + b_n \ast B) + (a_n + A \ast b_n)\sqrt B(A+B​)n+1=an+1​+bn+1​B​=(an​+bn​B​)∗(A+B​)=(A∗an​+bn​∗B)+(an​+A∗bn​)B​
